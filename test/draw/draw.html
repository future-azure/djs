COMMANDS:(Case insensitive, separated by space)
  FD x - Go Forward x pixels
  BK x - Go Back x pixels
  RT x - Turn Right x degrees
  LT x - Turn Left x degrees
  PU - Pen Up
  PD - Pen Down
  REPEAT n [ ... ] - Loop(Allowed Nested Loop)
  CL - Clear
  HOME - Move pen to center
  TO name args... [ ... ] - Define a precedure
  (args must start with ':', allowed call procedure in procedure)
  IF condition [ ... ] - Conditional statements
  (only '<','>','=' supported, conditions can be connected with '&'(AND), '|'(OR))
  STOP - Stop drawing(can be used as return in procedure)

SAMPLES:
  Triangle : REPEAT 3 [RT 120 FD 100]
  Rectangle : REPEAT 4 [FD 50 RT 90]
  Star : REPEAT 5 [FD 100 RT 144]
  Circle : REPEAT 60 [FD 10 RT 6]
  Circles : REPEAT 10 [RT 36 REPEAT 60 [FD 10 RT 6]]
  Spiral1 : TO SPIRAL :A [IF :A > 100 [STOP] FD :A RT 90 SPIRAL :A + 5] SPIRAL 5
  Spiral2 : TO SPIRAL :A :B [IF :A > 400 [STOP] FD :A RT 120 SPIRAL :A + :B :B + 1] SPIRAL 2 2
  Tree : TO TREE :A [FD :A IF :A < 50 [BK :A STOP] LT 20 TREE :A - 10 RT 40 TREE :A - 10 LT 20 BK :A] RT 90 PU FD 250 LT 180 PD TREE 100
